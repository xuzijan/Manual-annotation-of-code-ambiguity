# IAA 标注问卷

## 第一步：阅读标注指南

请先完整阅读 **[人工标注指南.md**](Manual annotation guidelines.md)，了解 8 类歧义标签的定义与使用规则，再进行答题。

## 第二步：开启网页答题

1. 在 VSCode 中打开集成终端（Windows / Mac 相同，或菜单 终端 → 新建终端）
2. 终端默认已在项目目录，直接输入 `node server.js` 并回车：
   ```
   node server.js
   ```
3. 浏览器访问：**http://localhost:3000**
4. 进入第一题前，会先进入「信息统计」页：填写问卷回答者英文名称，并选择要做的数据集编号（1-20）
5. 按提示完成题库中的标注

## 第三步：提交答案

答题完成后，请将 **`data/`** 目录下的问卷答案文件（`progress.json`）收集并提交给我。
